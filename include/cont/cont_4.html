스킵내비게이션(Skip Navigation)

스킵 네비게이션은 웹 표준을 준수하기 위해 그리고 장애인도 평등하게 웹을 사용하기 위한 웹 접근성을 위해 꼭 설정해야 하는 요소 중 하나입니다.
네비게이션 영역의 많은 링크들을 건너뛰고 바로 중요한 컨텐츠 정보를 탐색하기 위해 사용합니다.
<!-- 탭키를누르면 메뉴바로가기로 이동 : 시각장애인들을 위한 웹표준 방법 -->
<!-- HTML -->
<div id="skip">
    <a href="#">대쉬보드 바로가기</a>
    <a href="#">관리 바로가기</a>
    <a href="#">보고서 바로가기</a>
    <a href="#">통계분석 바로가기</a>
    <a href="#">설정 바로가기</a>
</div>
/* CSS */
#skip {position: relative;}
#skip a{
    position: absolute; left: 0px; top:-50px; 
    border:1px solid #fff; color: #fff; background: #333;
    line-height: 30px; width: 140px; text-align: center; z-index: 1000;
}
#skip a:active,
#skip a:focus{
    top:0;
}
