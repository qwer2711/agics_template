** 비주얼스튜디오 단축키
 
* Shift + Alt + F 코드정리
* Shift + Alt + A 여러줄 주석 설정, 해제
* Shift + Alt + ↑, ↓  줄 복사
 
* Alt + ↑, ↓  줄 이동
* Alt + 마우스  다중커서 포인터
* Alt + Ctrl + ↑, ↓  다중 커서 생성
 
* Ctrl + f  검색
* Ctrl + ], [  줄 들여쓰기, 내어쓰기
* Ctrl + X  줄 잘라내기(블럭 지정 없이)
* Shift + Ctrl  + M  문제 탭 토글
* Ctrl + Enter  아래에 신규 행 삽입
* Ctrl + Page Up, Page Down  탭 이동
