대체 텍스트

눈으로 화면을 볼 수 없는 경우, 이미지에 대한 설명을 대체 텍스트로 입력하여 스크린리더를 통해 정보를 인식하게 도와줍니다.
텍스트 아닌 콘텐츠는 그 의미나 용도를 인식할 수 있도록 대체 텍스트를 제공해야합니다.
/* 대체텍스트 */
/* 의미있는 이미지의 대체 텍스트를 제공하는 경우 */
.ir_pm {display: block; overflow: hidden; font-size: 0; line-height: 0; text-indent: -9999px;}
/* 의미있는 이미지의 이미지가 없어도 대체 텍스트로 보여주고자 할 때 */
.ir_wa {display: block; overflow: hidden; font-size: 0; position: relative; z-index:-1;
    width: 100%; height: 100%;
}
/* 대체 텍스트가 아닌 접근성을 위한 숨김 텍스트 제공 */
.ir_su{overflow: hidden; position: absolute; width: 0; height: 0; line-height: 0; text-indent: -9999px;}
