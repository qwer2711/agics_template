ellipsis

Single Line Ellipsis (하나의 줄일 경우)

생략부호의 예시입니다. 이 문장이 300px의 길이에 도달하면, 텍스트는 끊어집니다.

Multiple Line Ellipsis (여러줄일 경우)

줄이 여러줄일 경우에 줄임말의 예입니다. 생략 부호가 적용되기 전에 라인의 수를 설정하고, CSS를 약간 변경하고, 우리가 원하는 라인 수에 도달하면 생략 부호가 효력을 발휘합니다.
소스코드
<!-- HTML -->
<div class="parent">
    <h2>Single Line Ellipsis (하나의 줄일 경우)</h2>
    <div class="ellipsis">
        생략부호의 예시입니다 . 이 문장이 300px의 길이에 도달하면, 텍스트는 끊어집니다.
    </div>
    <h2>Multiple Line Ellipsis (여러줄일 경우)</h2>
    <div class="block-ellipsis">
        줄이 여러줄일  경우에 줄임말의 예입니다 . 생략 부호가 적용되기 전에 라인의 수를 설정하고, CSS를 약간 변경하고, 우리가 원하는 라인 수에 도달하면 생략  부호가 효력을 발휘합니다.
    </div>
</div>
/* CSS */
.parent {
    width: 300px;
}
.ellipsis {
  white-space: nowrap;
  text-overflow: ellipsis;
  overflow: hidden;
}
.block-ellipsis {
  display: block;
  display: -webkit-box;
  max-width: 100%;
  height: 43px;
  margin: 0 auto;
  font-size: 14px;
  line-height: 1;
  -webkit-line-clamp: 3; /* 라인수 */
  -webkit-box-orient: vertical;
  overflow: hidden;
  text-overflow: ellipsis;
}
